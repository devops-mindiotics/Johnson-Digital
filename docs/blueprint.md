# **App Name**: EduCentral

## Core Features:

- RBAC Implementation: Implement role-based access control to manage user permissions and menu access based on roles (Super Admin, School Admin, Teacher, Student).
- User Authentication: Secure login functionality with mobile number and password, including password visibility toggle and a 'Forgot Password' flow with OTP validation.
- Dashboard Customization: Dynamically generate dashboards based on user roles, displaying relevant statistics, graphs, and quick access menus for Super Admin, School Admin, Teacher, and Student.
- School Management (Super Admin): Enable Super Admins to manage schools, including onboarding, editing school details, and managing licenses (Teachers and Students).
- User Management: Allow Super Admins and School Admins to manage users, including creating individual or bulk users via CSV upload.
- Content Delivery: Teachers create and students view Subject lessons including videos, PDFs (course book, workbook, lesson plan, answer key), PPTs, etc.. 
- Personalized Notifications: Generate in-app notifications based on user role to keep up to date on tasks (ie: homeworks) or information (ie: notice board)

## Style Guidelines:

- Primary color: Deep Blue (#4169E1) for a sense of trust and security, aligning with the educational context.
- Background color: Light Blue (#E6E8FF) for a calm and approachable feel.
- Accent color: Light Purple (#B19CD9) for highlighting interactive elements and calls to action.
- Body and headline font: 'Nunito' (sans-serif) for a modern, friendly, and readable text; note: currently only Google Fonts are supported.
- Mobile-responsive design with clear, intuitive navigation and consistent layout across all screens to ensure usability on different devices.
- Use clean, modern icons to represent different menu items and actions, ensuring they are easily recognizable and accessible.
- Subtle animations and transitions to enhance user experience, such as smooth page transitions and loading indicators.
